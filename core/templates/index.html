Projects

  • A Movie streaming service platform: Netchlix.com

    • Netchlix is a movie streaming service platform built majorly with Django and Tailwindcss. The django-allauth authentication module is incorporated into the project to handle registration and authentication.   • A Social Media Platform: socialsbyjonah.social

    • Socialsbyjonah is a social media web app built with Django and Tailwindcss. The theme of the webpages can be toggled between light and dark modes. On successfully signing up, users receive an email welcoming them to the platform. A user can change password and reset password via email. Users can follow/unfollow other accounts on the platform, make/delete/like/download posts. A user can also search for users on the platform where the result of the search is based on the user's input. Logged In users can also view a list of accounts they follow and also those that follow them. The project is on a live Postgres database served by Amazon Web Services'(AWS) Relational Database Service(RDS). Uploaded files are served by the AWS S3 bucket service for best security.
