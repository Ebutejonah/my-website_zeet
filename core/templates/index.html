Projects

  • A fully functioning Netflix Clone Web App: Netchlix.com

    • Netchlix is a Netflix clone web app built majorly with Django and Tailwindcss. The django-allauth authentication module is incorporated into the project to handle registration and authentication. On creating an account, a user would also create a profile where "Maturity Level" is selected. Movies are available to each profile based on the maturity level selected. The project is on a live Postgres database served by Amazon Web Services'(AWS) Relational Database Service(RDS). Video files are served by the AWS S3 bucket service for best security.

  • A fully functioning Social Media Platform: socialsbyjonah.social

    • Socialsbyjonah is a social media web app built with Django and Tailwindcss. The theme of the webpages can be toggled between light and dark modes. On successfully signing up, users receive an email welcoming them to the platform. A user can change password and reset password via email. Users can follow/unfollow other accounts on the platform, make/delete/like/download posts. A user can also search for users on the platform where the result of the search is based on the user's input. A Logged In user can also view a list of accounts he/she follows and also those that follow him/her. The project is on a live Postgres database served by Amazon Web Services'(AWS) Relational Database Service(RDS). Uploaded files are served by the AWS S3 bucket service for best security.
